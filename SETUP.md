# ZHTP Setup Guide - From Zero to Earning Tokens

## 🚀 **Quick Setup (2 Minutes)**

### Method 1: GUI Installer (Easiest)
1. **Download**: Clone or download ZHTP to your computer
2. **Install**: Run `.\app_installer\build-installer.bat` (Windows)
3. **Launch**: The installer creates ZHTP-Installer.exe
4. **Setup**: Follow the GUI to create wallet and choose node type
5. **Earn**: Start earning ZHTP tokens immediately!

### Method 2: Command Line (Developers)
1. **Launch**: Run `.\run-zhtp.ps1`
2. **Verify**: Run `.\simple-verify.ps1` 
3. **Test**: Run `.\final-security-test.ps1`
4. **Browse**: Browser opens automatically to http://localhost:4000
5. **Earn**: You're now earning ZHTP tokens!

## 🔧 **Node Types**

### Validator Node
- **Earnings**: High (consensus rewards)
- **Requirements**: Good internet, reliable uptime
- **Stake**: Higher stake required
- **Role**: Validate transactions and blocks

### Router Node  
- **Earnings**: Medium (routing rewards)
- **Requirements**: Good bandwidth
- **Stake**: Medium stake required
- **Role**: Route packets through the network

### Storage Node
- **Earnings**: Steady (storage rewards)
- **Requirements**: Available disk space
- **Stake**: Lower stake required
- **Role**: Store data for the network

### Raspberry Pi Node
- **Earnings**: Basic (participation rewards)
- **Requirements**: Raspberry Pi 4+
- **Stake**: Minimal stake required
- **Role**: Support network decentralization
